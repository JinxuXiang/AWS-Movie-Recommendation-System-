We used the movie review dataset from MovieLens 1M datasert. The data set contains 1 million ratings from 6000 users on 4000 movies, released on 2/2003.

Based on the existing data set, we built a DynamoDB on AWS. This database contains several tables. For example, a Movie table contains all movie information, a User table contains all user information, a Rating table contains the ratings, a History table contains the user's watching history.

For each existing users in the dataset, we created a random userid and password for them. We also built a website to sign in or sign up accounts. It is possible for new users to join the platform. There also have a website that recommend movies, recommend new friends and groups based on users' rating and watching history.

We built the application of chatbot on movie recommendation. Based on the conversations between users and chatbot, we anticipate to reach a higher accuracy on user’s interest and therefore recommend movies that best match user’s preference.
